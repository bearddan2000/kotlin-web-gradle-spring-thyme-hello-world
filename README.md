# kotlin-web-gradle-spring-thyme-hello-world

## Description
A springboot web app with thymeleaf support.

## Tech stack
- kotlin
- gradle

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
